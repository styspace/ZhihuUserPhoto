## 基于Spring和Spring MVC来抓取知乎用户的数据
### 原理
- 首先在页面输入登录所需的验证码，默认是手机号登录(手机号和密码都在配置文件里面)
- 登录之后访问首页，将服务器返回的cookie信息序列化保存在本地，这样，下次就不需要登录，直接开始爬数据
- 登录之后，以个人首页关注者URL作为开始爬取的页面，然后将用户的信息保存到数据库中

>跑了5天，抓取了15W左右个用户信息，

### 难点
- HttpClient方法的使用
- spring中的多线程问题
